## 📚 What I Studied Today
- Topic: Spring Boot — Inversion of Control (IoC) & Dependency Injection (DI)
  - Resource: Personal notes
  - What I understood: IoC means Spring creates and manages objects (beans) in the IoC container rather than the application code instantiating them. DI is the mechanism by which Spring injects those dependencies into classes (constructor, setter, or field injection). Use `@Component` (or `@Service`) to register beans, `@Autowired` to request injection, and `@Qualifier` / `@Primary` to disambiguate when multiple beans match.
