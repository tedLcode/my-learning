## 📚 What I Studied Today
- Topic: BDI — Session 3 notes
  - `security_from_EKS.py`: vault agent mounts secrets into container at `/vaultdata/`; functions `get_secrets_raw()`, `get_secrets_value()`, `get_secrets_file()` provide DB creds, RSA keys, and arbitrary files. `SECRETS_PATH_CONTAINER = "vaultdata"` in `constant.py`.
  - `database_adapter.py` / `postgresql_database_adapter.py`: uses `psycopg2`; `get_connection()` builds connection from vault `dbcreds` and env vars; `get_data(sql)` returns rows as uppercase-key dicts; `save(sql)` runs INSERT/UPDATE and commits.
  - `s3_utilities.py`: uses `boto3` with retries; functions include `list_s3_objects`, `read_s3_object_bytes`, `copy_s3_file`, `move_s3_file`, `delete_s3_file`, `get_s3_object_info`. `list_s3_objects` returns direct children only; pattern matching via `fnmatch`.
  - `common_utilities.py`: helpers `generate_task_status()`, `populate_metadata_with_values()`, `metadata_to_dataclass()`, `generate_file_name()`; these manage pass/fail flags, replace `$VAR` and date placeholders, convert DB rows to dataclasses, and build target filenames.
